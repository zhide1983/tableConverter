🔄 转换格式示例

🌐 HTML 格式

完整的 HTML 表格，保留合并单元格和样式：

<table>
  <thead>
    <tr>
      <th>部门</th>
      <th>季度</th>
      <th>销售额</th>
      <th>增长率</th>
      <th>备注</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td rowspan="2" style="background: linear-gradient(135deg, #e3f2fd 0%, #f3e5f5 100%); font-weight: 500;">销售部</td>
      <td>Q1</td>
      <td>¥1,200,000</td>
      <td>+15%</td>
      <td>超额完成</td>
    </tr>
    <tr>
      <td>Q2</td>
      <td>¥1,350,000</td>
      <td>+12%</td>
      <td>稳步增长</td>
    </tr>
  </tbody>
</table>

📝 Markdown 格式

标准 Markdown 表格语法：

| 部门 | 季度 | 销售额 | 增长率 | 备注 |
| --- | --- | --- | --- | --- |
| 销售部 (1×2) | Q1 | ¥1,200,000 | +15% | 超额完成 |
|  | Q2 | ¥1,350,000 | +12% | 稳步增长 |
| 市场部 (1×2) | Q1 | ¥800,000 | +8% | 达成目标 |
|  | Q2 | ¥920,000 | +15% | 表现优异 |

🧵 Textile 格式

Textile 标记语言表格：

|_. 部门|_. 季度|_. 销售额|_. 增长率|_. 备注|
|/2. 销售部|Q1|¥1,200,000|+15%|超额完成|
|Q2|¥1,350,000|+12%|稳步增长|
|/2. 市场部|Q1|¥800,000|+8%|达成目标|

⚡ 使用流程演示

步骤1: 在 Excel 中选择表格区域并复制 (Ctrl+C)
步骤2: 打开 TableClip 工具页面
步骤3: 点击 "粘贴表格数据" 按钮
步骤4: 选择目标格式 (HTML/Markdown/Textile/RST)
步骤5: 复制生成的代码或下载文件

🎯 特色功能

  • 🔄 智能解析: 自动检测表格结构和合并单元格
  • 🎨 样式保留: 可选择保留原始格式和样式
  • 📱 响应式支持: 生成移动端友好的表格代码
  • ⚡ 实时预览: 即时查看转换效果
  • 💾 多种导出: 支持复制到剪贴板或下载文件
  • 🔒 隐私安全: 纯前端处理，数据不上传
